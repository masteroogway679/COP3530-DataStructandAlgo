# Data Structures & Algorithms (COP 3530)

This repository contains my coursework, programming assignments, and projects completed for COP 3530: Data Structures & Algorithms. The course focuses on implementing and analyzing fundamental data structures, algorithm efficiency, and object-oriented programming concepts in C++.

## Topics Covered
- Algorithm analysis and Big-O notation
- Abstract Data Types (ADTs)
- Arrays and Linked Lists
- Stacks and Queues
- Templates and Generic Programming
- Object-Oriented Programming (Inheritance & Composition)
- Searching and Sorting Algorithms
- Recursion
- Trees and Graph fundamentals
